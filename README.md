# Claude Code Skills

Auto-generated agent skills from the fastest-growing **Claude Code** open source projects on GitHub, by [ara.so](https://ara.so).

A bot scans GitHub every 30 minutes for trending repos in the Claude Code niche, ranks them by stars/day, and writes an installable `SKILL.md` for the top new one.

## Install all skills

```bash
npx skills add Aradotso/claude-code-skills
```

## Install a specific skill

```bash
npx skills add Aradotso/claude-code-skills --skill <skill-name>
```
